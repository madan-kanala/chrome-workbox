Strategies

This is a simple class used to namespace the supported caching strategies in sw-lib.

You would never access this class directly but instead use with swlib.strategies.<Strategy Name>.

Instance Methods

this.cacheFirst(options) → CacheFirst

A cache first run-time caching strategy.

Example
const cacheFirstStrategy = goog.swlib.strategies.cacheFirst();

goog.swlib.router.addRoute('/styles/*', cacheFirstStrategy);
Parameters
options RuntimeStrategyOptions (Optional)

To define any additional caching or broadcast plugins pass in option values.
Returns CacheFirst

A CacheFirst handler.
