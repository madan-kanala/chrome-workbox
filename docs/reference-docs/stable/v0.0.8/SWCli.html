SWCli

This class is a wrapper to make test easier. This is used by ./bin/index.js to pass in the args when the CLI is used.

Instance Methods

this._generateSW() → Promise

This method will generate a working service worker with a file manifest.

Returns Promise

The promise returned here will be used to exit the node process cleanly or not.

this._getRootOfWebApp() → Promise.<string>

This method requests the root directory of the web app. The user can opt to type in the directory OR select = require(a list of directories in the current path.

Returns Promise.<string>

Promise the resolves with the name of the root directory if given.

this.handleCommand(command, args, flags) → Promise

If a command is given in the command line args, this method will handle the appropriate action.

Parameters
command string

The command name.
args object

The arguments given to this command.
flags object

The flags supplied with the command line.
Returns Promise

A promise for the provided task.

this.handleFlag(flags) → Promise

If there is no command given to the CLI then the flags will be passed to this function in case a relevant action can be taken.

Parameters
flags object

The available flags = require(the command line.
Returns Promise

returns a promise once handled.
